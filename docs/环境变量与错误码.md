# 环境变量与错误码

## 1. 环境变量

在 `backend/.env` 中配置：

| 变量 | 必需 | 默认值 | 说明 |
|------|------|--------|------|
| `DATABASE_URL` | 是 | `sqlite+aiosqlite:///./graphchat.db` | 数据库连接串 |
| `OPENAI_API_KEY` | 否 | — | OpenAI API 密钥（使用 MockLLM 时可不填） |
| `OPENAI_MODEL` | 否 | `gpt-4` | 默认模型 |
| `LLM_PROVIDER` | 否 | `mock` | LLM 提供者：`openai` 或 `mock` |
| `CORS_ORIGINS` | 否 | `http://localhost:3000` | 允许的 CORS 来源（逗号分隔） |
| `APP_ENV` | 否 | `development` | 运行环境：`development` / `production` |
| `LOG_LEVEL` | 否 | `INFO` | 日志级别 |

## 2. 错误码体系

### 2.1 通用错误

| 错误码 | HTTP | 说明 |
|--------|------|------|
| `SUCCESS` | 200 | 成功 |
| `VALIDATION_ERROR` | 400 | 请求参数校验失败 |
| `NOT_FOUND` | 404 | 资源不存在 |
| `INTERNAL_ERROR` | 500 | 服务器内部错误 |

### 2.2 对话模块 `CONV_*`

| 错误码 | HTTP | 说明 |
|--------|------|------|
| `CONVERSATION_NOT_FOUND` | 404 | 对话不存在 |
| `CONVERSATION_ALREADY_ARCHIVED` | 400 | 对话已归档，无法操作 |
| `CONVERSATION_HAS_CHILDREN` | 400 | 对话存在子分支，无法删除 |
| `INVALID_PARENT` | 400 | 无效的父对话 ID |
| `TREE_DEPTH_EXCEEDED` | 400 | 超过最大树深度限制 |

### 2.3 消息模块 `MSG_*`

| 错误码 | HTTP | 说明 |
|--------|------|------|
| `MESSAGE_NOT_FOUND` | 404 | 消息不存在 |
| `MESSAGE_EMPTY_CONTENT` | 400 | 消息内容不能为空 |
| `MESSAGE_TOO_LONG` | 400 | 消息内容超过长度限制 |
| `FORK_TEXT_TOO_SHORT` | 400 | 分叉选中文本过短（< 1 字符） |
| `FORK_TEXT_TOO_LONG` | 400 | 分叉选中文本过长（> 100 字符） |
| `FORK_FROM_NON_ASSISTANT` | 400 | 只能从 AI 回答中分叉 |

### 2.4 标注模块 `ANN_*`

| 错误码 | HTTP | 说明 |
|--------|------|------|
| `ANNOTATION_NOT_FOUND` | 404 | 标注不存在 |
| `ANNOTATION_INVALID_RANGE` | 400 | 标注范围无效（offset 超出消息长度） |
| `ANNOTATION_DUPLICATE` | 400 | 该位置已存在标注 |
| `ANNOTATION_NO_SUGGESTIONS` | 400 | 标注必须包含至少一个建议 |

### 2.5 Agent 模块 `AGENT_*`

| 错误码 | HTTP | 说明 |
|--------|------|------|
| `AGENT_SUGGEST_FAILED` | 502 | 生成分叉建议失败（LLM 错误） |
| `AGENT_EXPLORE_FAILED` | 502 | 自主探索失败 |
| `AGENT_EXPLORE_ALREADY_RUNNING` | 400 | 该分支已在探索中 |
| `AGENT_MERGE_FAILED` | 502 | 综合结论失败 |
| `AGENT_MERGE_NO_SOURCES` | 400 | 未选择任何源分支 |
| `AGENT_MERGE_INVALID_TARGET` | 400 | 无效的合并目标 |
| `LLM_PROVIDER_ERROR` | 502 | LLM 服务调用失败 |
| `LLM_RATE_LIMITED` | 429 | LLM 服务限流 |

---

## 3. 数据库连接串示例

```bash
# 开发环境（SQLite）
DATABASE_URL=sqlite+aiosqlite:///./graphchat.db

# 生产环境（PostgreSQL）
DATABASE_URL=postgresql+asyncpg://user:pass@localhost:5432/graphchat
```
