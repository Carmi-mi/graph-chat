# Graph Chat - 项目实现计划

## 项目概述

**Graph Chat** 是一款基于图结构的协同研究 Agent，通过非线性对话方式帮助用户深度探索复杂话题。核心差异化在于树状/网状对话结构，用户可自由选择深入方向，Agent 辅助标注推荐方向，实现人机协同的深度研究。

### M2: 接口文档完善 (Week 1-2) ✅ 已完成

#### 2.1 对话管理接口
| 方法 | 路径 | 请求 | 响应 | 说明 |
|------|------|------|------|------|
| POST | /api/conversations | `{name: string}` | `Conversation` | 创建对话 |
| GET | /api/conversations/{id} | - | `ConversationWithTree` | 获取对话（含树结构） |
| GET | /api/conversations | - | `ConversationListResponse` | 获取对话列表 |
| PUT | /api/conversations/{id} | `{name, status}` | `Conversation` | 更新对话 |
| DELETE | /api/conversations/{id} | - | `{success: boolean}` | 删除对话 |

#### 2.2 消息管理接口
| 方法 | 路径 | 请求 | 响应 | 说明 |
|------|------|------|------|------|
| POST | /api/messages | `{conversationId, role, content}` | `Message` | 发送消息 |
| GET | /api/messages/{convId} | - | `Message[]` | 获取对话消息列表 |
| POST | /api/messages/{id}/fork | `{selectedText, suggestion?}` | `Conversation` | 从消息分叉 |

#### 2.3 标注接口
| 方法 | 路径 | 请求 | 响应 | 说明 |
|------|------|------|------|------|
| POST | /api/annotations | `{messageId, text, suggestions}` | `Annotation` | 创建标注 |
| GET | /api/annotations/{msgId} | - | `Annotation[]` | 获取消息标注 |
| DELETE | /api/annotations/{id} | - | `{success: boolean}` | 删除标注 |

#### 2.4 Agent 接口
| 方法 | 路径 | 请求 | 响应 | 说明 |
|------|------|------|------|------|
| POST | /api/agent/suggest | `{messageId}` | `{suggestions, count}` | 获取分叉建议 |
| POST | /api/agent/auto-explore | `{branchId, maxDepth, parallel}` | `{taskId}` | 启动自主探索 |
| GET | /api/agent/status/{convId} | - | `{branches: BranchStatus[]}` | 获取探索状态 |
| POST | /api/agent/merge | `{targetId, sourceIds, keepOption}` | `{conclusion}` | 综合结论 |
